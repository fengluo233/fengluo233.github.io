黑苹果使用感想

发表于 2020-04-11 阅读次数： Valine：
本文字数： 980 阅读时长 ≈ 2 分钟

前言

总是耳闻黑苹果有多么多么好用，这次总算是尝试了一下，发现……

还真好用！

坎坷过程

安装的时候的确很麻烦，前前后后折腾了好久，上网查了不少资料都没有头绪，最后在前辈们的各种文档和遇到什么问题解决什么问题的精神，总算是勉强装好了。但还存在类似 WiFi驱动不能用，触摸板适配存在问题等小麻烦，总的来说不影响使用。但是如果没有现成的efi文件的话真的是十分十分十分麻烦，自己写配置对我这种小白来说还是太艰难了！

具体使用

装好之后心情十分激动，总的来说体验感还是十分不错的，从安装软件等等小细节上面可以说体验是十分舒适的，但我认为简洁的操作也带来了一些问题：可定制性不够强。安装软件不能进行分类让软件多了之后的管理会变得不太容易。而且可能是刚开始使用的原因，苹果的文件结构不太熟悉，终端用起来就有点难受，找东西半天找不到，反正挺难受的。

但总的来说，优点还是要大过缺点的，至少在日常使用的软件方面，QQ，微信，网易云音乐，telegram，这些聊天与娱乐软件的使用很好的体现了苹果在设计方面下的功夫，无论是通知提醒还是快速回复，都能带来很愉悦的体验。点名批评QQ UWP 版本，设计有大问题，上传文件失败的问题这么久了还没修复。

写代码方面由于刚上手没多久，生产力工具都在那边还没迁移，现在这边只有一个 VSCode，反正用起来挺顺手，用VSCode写个C语言小程序还要折腾 GCC 就™离谱。
其他方面等到迁移了之后再说，刚好最近要整一个Python的东西，现在就拿他写了，试试看。jetbrain家的东西等用了再说。

感想

自从苹果使用英特尔架构之后，源源不断的开发者涌入到黑苹果的行列中，从当年的 Tiger 到现在的 Catalina ，轮子让黑苹果变得越来越简单，感谢 Clover 让我只用简单了解config.plist，也就是和自己电脑型号相关的配置就可以安装上黑果，相对成熟的技术也让出现问题的解决方案变得多元而简便，其实说 OS X 好用，能有多好用？还不是写不出代码个操作系统，但是折腾他的过程反而是让我享受的，在这之中必须去了解系统的引导方法，引导过程并尝试添加，在遇到驱动不能使用的时候想办法去寻找各式各样的驱动，安装好之后感受各种不同的系统的细微但有趣的差别，现在想想，我是真的喜欢 OS X 吗？不是的，就是想折腾一下而已。

而且 Mac 那么那么那么贵怎么买的起啊！！
