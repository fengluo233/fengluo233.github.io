开始设置

在命令行执行下面的命令

然后将你要设置的terminal图标拖到这个目录里边，这里提供一个资源。

检查文件是否有startingDirectory属性

检查 Windows terminal 的 profile.json 中是否有 startingDirectory 属性的设置，如果有，则需要将其删除。不删除的话，打开的初始位置是固定的，不能做到在当前路径打开。
